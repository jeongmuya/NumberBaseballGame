# ⚾️ 숫자 야구 게임

## 프로젝트 소개
Swift로 구현한 숫자 야구 게임입니다.

## 구현 기능
### Lv1 (필수)
- [x] 1-9까지의 서로 다른 임의의 수 3개 생성
- [x] 랜덤 숫자 생성 기능

### Lv2 (필수)
- [x] 사용자 입력 받기
- [x] 입력값 유효성 검사
- [x] 스트라이크와 볼 판정
- [x] 게임 종료 조건 구현

## 실행 화면
< 게임을 시작합니다 >
숫자를 입력하세요
435



## 프로젝트 구조
- BaseballGame.swift: 게임의 주요 로직 구현
- main.swift: 게임 실행

## 실행 방법
1. Xcode에서 프로젝트 열기
2. Command Line Tool 프로젝트로 실행
3. 콘솔에서 숫자 입력하며 게임 진행
